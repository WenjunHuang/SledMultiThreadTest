Sled Multi-threaded test

USAGE:
    SledMultithreadTest <INPUT> -m <mode> -t <threads>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -m <mode>           Test write only
    -t <threads>        threads count

ARGS:
    <INPUT>    Sled directory
